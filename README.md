Kali Nethunter Tools Installer

NOTE:
This is for Android running an ARM64 processor.

Intented to allow Kali Nethunter tools to be installed on your own lightweight Debian installation
like a Debian minimal. 

REQUIREMENTS:
Install Debian minimal by installing TERMUX.
Have a rooted device.
Download this script.
Run it.
